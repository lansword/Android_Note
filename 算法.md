## 1、快速排序

```java
public void fastSort(int[] a[],int left,int right){
  if(left >= right){
    return;
  }
  int p = a[left];
  int i = left,j = right;
  while(i < j){
    while(i < j && a[j] >= p){
      j--;
    }
    if(i < j){
      a[i++] = a[j];
    }
    while(i < j && a[i] <= p){
      i++;
    }
    if(i < j){
      a[j--] = a[i];
    }
  }
  a[i] = p;
  fastSort(a,left,i-1);
  fastSort(a,i+1,right);
}
```

## 2、冒泡排序

```java
    /**
     * 冒泡排序
     * @param arr
     */
    public void bubbleSort(int[] arr) {
        for (int i = arr.length - 1; i > 0; i--) { //每次需要排序的长度
            for (int j = 0; j < i; j++) { // 从第一个元素到第i个元素
                if (arr[j] > arr[j + 1]) {
                    int t = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = t;
                }
            }
        }
    }
```
